Kenya Electoral Wards API

Introduction

The Kenya Electoral Wards API provides the user with the names of all the wards in a particular county or counties based on the URL parameters provides. With this payload, the user can choose the data they want to consume in their app/website.

Kenya has 47 counties. Each county is divided into constituencies and each constituency is divided into wards. As such, wards are the smallest electoral divisions in the country and closest the citizens can get government services.

Overview

The most important thing you need to know as a developer is the syntax I've used when adding the names of the counties to the database. If you do not write the names exactly as in the following list, you will get an empty response.

Make sure you follow the capitalizations and dashes that I've used to the letter.

very important: There are counties with 2 names. Examples are "West Pokot", "Homa Bay" and "Uasin Gishu". When performing a GET request on such counties, you need to separate the two names with a "+".

Example:

/api/wards?county=West+Pokot /api/wards?county=Homa+Bay
/api/wards?county=Uasin+Gishu

Authentication

There is no authentication required to use this API. I will keep an eye on the number of requests to the API, and if it hits 10000 requests per month, I will enable authorization.

This will help me know who is using the API and for what purpose.

Error Codes

402 - You will get this error when you try to perform a POST request without providing one or all of the following in the request body:

  1. Ward name
  2. Constituency name
  3. County Name

In all other cases, you will get an empty response, showing that you have entered a county that does not exist in Kenya, or you have a typo in the request parameter.

Rate limit

As of this writing, I have not imposed any rate limit on the use of this API.

GET https://frozen-basin-45055.herokuapp.com/api/wards?county=Nairobi

This returns the names of all the wards in "Nairobi" county. You can replace "countyName" with the name of any of the 47 counties in the country.

GET https://frozen-basin-45055.herokuapp.com/api/wards?county=Nairobi&county=Mombasa

This returns the names of all the wards in Nairobi and Mombasa counties. You can replace "Nairobi" or "Mombasa" with the name of any of the 47 counties in the country. You can display more counties. Just separate them with "&" as shown in this request URL example.

POSThttps://frozen-basin-45055.herokuapp.com/api/wards

This adds a new ward to the wards database. You need to provide the name of the ward on the "name" property, the name of the constituency on the "constituency" property and the name of the county on the "county" property.

PUT https://frozen-basin-45055.herokuapp.com/api/wards/:id

This updates data about the ward whose id is specified in the "id" request parameter.

DELETE https://frozen-basin-45055.herokuapp.com/api/wards/:id

This deletes the ward whose id is specified in the "id" request parameter.
