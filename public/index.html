Kenya Electoral Wards API

Introduction

The Kenya Electoral Wards API provides the user with the names of all the wards in a particular county or counties based on the URL parameters provides. With this payload, the user can choose the data they want to consume in their app/website.

Kenya has 47 counties. Each county is divided into constituencies and each constituency is divided into wards. As such, wards are the smallest electoral divisions in the country and closest the citizens can get government services.
